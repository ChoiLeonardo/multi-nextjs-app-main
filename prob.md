## 1주차: 중학교 확률/통계 기본기 다지기 (약 7일) 

고등학교 과정을 시작하기 전, 필수적인 중학교 개념을 빠르게 복습하여 토대를 튼튼히 합니다. 

| 목표                | 핵심 주제                                                            | 학습 자료/방법                 | 학습 시간    |
| ----------------- | ---------------------------------------------------------------- | ------------------------ | -------- |
| **기본 용어 및 계산 숙달** | **1. 통계 기본:** 평균, 중앙값, 최빈값, 분산, 표준편차                             | 중학교 3-2 수학 교과서 또는 기본 개념서 | 일일 1~2시간 |
|                   | **2. 확률 기본:** 경우의 수 계산, 수학적 확률, 통계적 확률, 확률의 기본 성질                | 중학교 3-2 수학 교과서 또는 기본 개념서 | 일일 1~2시간 |
| **학습 방법**         | 개념을 빠르게 읽고, 필수 예제와 쉬운 문제 위주로 풀어보며 계산 실수 없이 빠르게 정답을 찾는 연습에 집중합니다. |                          |          |

## 2주차: 고등학교 '확률과 통계' - 경우의 수 및 확률 (약 7~10일) 

고등학교 '확률과 통계' 과목의 첫 단원인 '경우의 수'와 '확률'의 핵심 개념을 익힙니다. 

| 목표            | 핵심 주제                                                                              | 학습 자료/방법                             | 학습 시간    |
| ------------- | ---------------------------------------------------------------------------------- | ------------------------------------ | -------- |
| **순열과 조합 이해** | **1. 경우의 수:** 합의 법칙, 곱의 법칙, 순열 (𝑛𝑃𝑟), 조합 (𝑛𝐶𝑟)                               | 고등학교 '확률과 통계' 개념 기본서 (예: 개념원리, 정석 등) | 일일 2~3시간 |
| **확률의 심화 개념** | **2. 확률:** 조건부 확률, 사건의 독립과 종속, 독립시행의 확률                                            | 고등학교 '확률과 통계' 개념 기본서                 | 일일 2~3시간 |
| **학습 방법**     | 중학교와 달리 복잡한 상황 판단이 요구됩니다. 공식을 암기하기보다는 각 공식이 어떤 상황에 적용되는지 예제를 통해 충분히 이해하는 것이 중요합니다. |                                      |          |

## 3주차: 고등학교 '확률과 통계' - 통계 (약 7~10일) 

확률 개념을 바탕으로 통계의 핵심 내용을 학습합니다. 이 부분이 중학교 과정과의 가장 큰 차이이며 고등학교 통계의 기반이 됩니다. 

| 목표                | 핵심 주제                                                           | 학습 자료/방법             | 학습 시간    |
| ----------------- | --------------------------------------------------------------- | -------------------- | -------- |
| **확률 분포 이해**      | **1. 확률 분포:** 확률 변수, 이산확률변수와 분포, 이항분포                           | 고등학교 '확률과 통계' 개념 기본서 | 일일 2~3시간 |
| **연속 확률 및 정규 분포** | **2. 정규 분포:** 연속확률변수와 분포, 정규분포, 표준정규분포                          | 고등학교 '확률과 통계' 개념 기본서 | 일일 2~3시간 |
| **학습 방법**         | 확률 분포표 작성, 기댓값과 분산 계산, 정규분포표 읽는 법 등 새로운 개념이 많으므로 꼼꼼한 학습이 필요합니다. |                      |          |

## 4주차: 심화, 복습 및 통계적 추정 맛보기 (약 7일) 

배운 내용을 정리하고, 고등학교 '확률과 통계'의 마지막 단원인 '통계적 추정'의 핵심 아이디어만 간략하게 살펴봅니다. 

| 목표           | 핵심 주제                                                                                     | 학습 자료/방법                       | 학습 시간    |
| ------------ | ----------------------------------------------------------------------------------------- | ------------------------------ | -------- |
| **전체 내용 복습** | 1~3주차 학습 내용 전반                                                                            | 오답 노트, 핵심 개념 요약 노트             | 일일 1~2시간 |
| **추정 개념 이해** | **통계적 추정 맛보기:** 표본 평균, 모평균의 추정 아이디어                                                       | 고등학교 '확률과 통계' 기본서의 '통계적 추정' 단원 | 일일 1~2시간 |
| **학습 방법**    | 헷갈렸던 개념(특히 조합 vs 순열, 조건부 확률 등)을 다시 확인하고, 마지막 단원은 복잡한 계산보다는 '왜 배우는가'라는 큰 그림을 이해하는 데 집중합니다. |                                |          |

## 성공적인 선행학습을 위한 Tip 

- **매일 꾸준히:** 1개월이라는 짧은 기간 동안 많은 내용을 소화하려면 매일 정해진 시간에 꾸준히 학습하는 것이 중요합니다.
- **개념 이해 우선:** 문제 풀이보다는 개념 이해에 집중해야 합니다. 선행학습 기간에는 심화 문제보다는 기본 예제와 유제 풀이를 통해 개념 적용 연습에 주력하세요.
- **인터넷 강의 활용:** 독학이 어렵다면, 고등학교 '확률과 통계' 개념 위주의 짧은 인터넷 강의를 활용하면 효율을 높일 수 있습니다.
- **오답 노트 작성:** 틀린 문제나 헷갈리는 개념은 반드시 오답 노트를 작성하고 주기적으로 복습하여 같은 실수를 반복하지 않도록 합니다.
